# 롯데월드 통합 관리자 페이지 (LotteWorld Admin)

본 프로젝트는 롯데월드 모바일 앱의 **회원·티켓·알림·지도/POI·운휴·공지/FAQ** 등  
운영 전반을 하나의 시스템에서 통합 관리할 수 있도록 구축된 **웹 기반 관리자 페이지**입니다.

관리자는 실시간 운영 현황을 확인하고, 콘텐츠·티켓·알림 등 주요 데이터를 효율적으로 관리하여  
롯데월드 전체 서비스의 운영 품질과 사용자 경험을 향상시킬 수 있습니다.

사이트: **https://media.seowon.ac.kr/s202011341/lotteworldadmin**
---

## 주요 기능

### 0. 관리자 로그인
- 관리자 회원가입 시 이메일 인증코드를 발송·검증하는 절차를 통해 계정을 활성화
- 인증 완료된 이메일만 회원가입 가능하도록 보안 강화
- 관리자 로그인은 이메일·비밀번호 기반으로 이루어지며 JWT 토큰을 통해 세션 관리
  
### 1. 홈 대시보드
- 실시간 방문자 수, 예매·회원·알림 요약지표 제공  
- 혼잡도, 최근 알림, 처리 대기 업무 등을 한눈에 확인 가능

### 2. 회원 관리
- 회원 목록 조회 / 검색 / 필터링  
- 회원 상세 정보 확인 (기본 정보, 예매/주문 내역 포함)  
- 회원 상태/등급 변경, 관리자 권한 설정·해제  
- 모달 기반 간편 편집 기능 제공

### 3. 티켓 예매 관리
- 전체 예매 현황 조회 및 조건 검색  
- 예매 상세 정보 확인 (티켓 정보, 결제 내역, 예약일 등)  
- 예매 상태 변경 및 운영 중 이슈 처리

### 4. 티켓 상품 등록/관리
- 신규 티켓 등록 또는 기존 티켓 불러오기  
- 티켓 유형에 따라 동적으로 변하는 입력 폼  
- 가격, 혜택, 옵션 정보 관리  
- 다양한 티켓 상품을 유연하게 구성 가능

### 5. 알림 관리 (FCM 연동)
- 관리자 페이지에서 작성한 알림을 **실시간 푸시 발송**  
- 알림 내용, 대상, 발송 방식 설정  
- 발송 이력 확인 및 예약 발송 기능 지원  
- 사용자 앱의 알림함 + FCM 푸시를 통한 이중 전달 구조

### 6. 지도 / POI / 경로 관리
- 어트랙션, 공연장, 음식점 등 POI 정보 관리  
- 위치, 운영 시간, 설명 등 상세 정보 수정  
- 지도 상 POI 좌표 및 경로(SVG) 등록/편집  
- 카테고리 관리(추가/수정/삭제)

### 7. 운휴 정보 관리
- 기상/점검/장애 등 콘텐츠 운휴 정보 등록·수정  
- 사용자 앱에 표시될 운휴 안내 문구 직접 관리  
- 알림 시스템과 연계하여 운휴/재개 알림 발송 가능

### 8. 공지사항 및 FAQ 관리
- 공지사항 등록·수정·삭제 및 노출 여부 관리  
- 공지 상세보기 제공  
- FAQ 카테고리 분류 및 질문/답변 관리

### 9. 시스템 설정
- 관리자 계정 관리  
- 알림 기본 설정 및 기타 운영 설정

## 기술 스택

### Frontend
- **Framework:** Next.js (App Router 기반)
- **Language:** TypeScript
- **Styling:** Tailwind CSS, PostCSS
- **구조:** 컴포넌트 기반 UI, Context 활용 전역 상태 관리

### Backend / Infra (연동)
- **API 서버:** Node.js + Express 기반 REST API  
- **Database:** MariaDB  
  - 사용자, 관리자, 티켓, 알림, 스케줄, POI 등 핵심 데이터 관리  
- **Email Verification:** Nodemailer 기반 이메일 인증코드 발송 및 검증 로직
- **Push Notification:** Firebase Cloud Messaging(FCM)

---

## 배포 방식 
- Next.js 정적 빌드(export) 결과물을 학교 웹 서버에 업로드하여 제공
- FileZilla를 활용한 정적 파일 기반 배포 방식
- 서버 환경에 맞춰 SPA 라우팅 처리
